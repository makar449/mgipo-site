# V8.24 — Restore Full Programs Catalog

## Цель
Вернуть полный каталог программ без отката ускорений сайта и без урезания контента ради GitHub Pages.

## Что исправлено
- Убрано ограничение `baseServices.slice(0, 8)` в `getPublishedServices()` для static export.
- Публичный каталог снова получает полный список программ из `baseServices`.
- Страница `/services/` сохраняет ленивый рендер карточек: сначала 24, далее кнопка «Показать ещё».
- Поиск и фильтры работают по полному массиву программ, но карточки не рендерятся все разом.
- Демо-админка получила полный lightweight-каталог программ и категорий из отдельного файла `src/features/demo-admin/admin-catalog.ts`.
- Ключ localStorage демо-админки обновлён до `mgipo_demo_admin_data_v5`, чтобы старые урезанные демо-данные не затирали полный список.

## Важно
Полный каталог означает, что static export создаёт больше страниц программ. GitHub Actions build может идти дольше, но публичный сайт больше не урезан до 8 программ.
