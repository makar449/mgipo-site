# V8.22 — Agile link, no-lag routing and motion refinement

## Что сделано

- В футер добавлена внешняя ссылка `сделано Agile business` на `https://www.agile-business-pro.com/`.
- Убрана агрессивная предзагрузка всех страниц одним потоком: теперь используется мягкий staged prefetch + prefetch по наведению/фокусу/нажатию.
- Отключён массовый Next Link prefetch в header/footer/buttons/admin-sidebar, чтобы сайт не грузил десятки route chunks сразу после открытия.
- Индикатор перехода больше не появляется мгновенно на каждый клик: добавлена короткая задержка, чтобы быстрые переходы не выглядели как вечная загрузка.
- Главная анимация усилена и ускорена через CSS transform/opacity без тяжёлого JS. Визуальная структура не менялась.
- Убрано content-visibility с hero и главного визуального блока, чтобы первый экран не выглядел как “просыпающийся” после ожидания.
- Адаптация 1024–1440px уточнена для ноутбуков: hero, 3D-блок и заголовки не должны заезжать друг на друга.

## Что не менялось

- Не менялась композиция сайта.
- Не удалялись анимации.
- Не менялись тексты, блоки, карточки и общий визуальный стиль.
- Админка осталась демо-режимом для GitHub Pages.
