# V8.21 Motion, Bundle Split & Laptop QA

## Сделано

1. Главная анимация возвращена без изменения дизайна: добавлены CSS-only transform/opacity-анимации для hero-блока, карточек, диплома, колец и подсветки.
2. Убран запрет `animation: none` для нового hero-motion слоя через более точечные классы. Декор не удалён.
3. Админка разделена по маршрутам на отдельные client chunks: страницы больше не импортируют общий `admin-pages.tsx` со всеми разделами.
4. Создан модульный слой `admin-lite`: общий shell/store отдельно, каждый раздел в отдельном файле.
5. Prefetch стал постепенным, а не массовым: сайт не пытается прогревать сразу все маршруты одним рывком.
6. Добавлена адаптация для ноутбуков 1024–1365px: hero-сетка, заголовок, визуал, карточка диплома, статистика и шапка ужаты без изменения композиции.
7. Визуально сайт не должен поменяться: изменения касаются скорости, маршрутизации, анимационного поведения и адаптации.

## Важное

GitHub Pages остаётся статическим хостингом. Демо-админка работает через localStorage. Для реальной общей админки позже нужен Supabase или другой backend.
