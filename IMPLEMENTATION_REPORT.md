## Команды проверки

```powershell

npx prisma generate
npm run lint
npm run check:content
npm run typecheck
npm run build
```

В текущей среде `typecheck/build` зависят от скачивания Prisma engine. При стабильном интернете локально должны запускаться после `npx prisma generate`.
