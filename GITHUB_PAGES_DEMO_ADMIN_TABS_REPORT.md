# V8.15 — routed demo admin tabs

Что изменено:

- Все разделы демо-админки открываются как отдельные маршруты GitHub Pages, а не только как внутренний предпросмотр.
- Боковое меню админки теперь использует реальные ссылки: `/admin/dashboard`, `/admin/home`, `/admin/services`, `/admin/categories`, `/admin/documents`, `/admin/legal`, `/admin/navigation`, `/admin/footer`, `/admin/contacts`, `/admin/leads`, `/admin/pages`, `/admin/seo`, `/admin/settings`, `/admin/users`, `/admin/audit`.
- Для каждого маршрута добавлен статический параметр, чтобы GitHub Pages отдавал страницу напрямую.
- В админке добавлены отдельные демо-разделы для программ, категорий, документов, правовых страниц, навигации, футера, SEO, пользователей и журнала действий.
- Публичная шапка сайта получила раскрывающиеся меню на desktop и вложенные пункты в мобильном меню.
- Демо-вход теперь ведёт на `/admin/dashboard`.

Ограничение:

- Это демо-админка для презентации на GitHub Pages. Данные сохраняются в `localStorage` текущего браузера. Для общей базы и реального сохранения всем пользователям нужен Supabase.
