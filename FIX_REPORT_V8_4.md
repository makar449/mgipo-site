# V8.4 Targeted QA Fix

Исправлены только пункты из последнего сообщения пользователя:

1. 3D-анимация на главной больше не создаёт большое пустое пространство перед собой: визуальный блок поднят вверх на desktop/tablet, уменьшена внутренняя высота, сохранена крупность и движение элементов.
2. Тексты в статистических карточках больше не должны переноситься по буквам/слогам: сетка статистики перестроена, глобальные небезопасные переносы переопределены точечно для hero/title/stat blocks.
3. Главный заголовок больше не должен ломаться как “квалификац / ии”: добавлен безопасный max-width, уменьшены агрессивные размеры на проблемных breakpoint’ах, отключены hyphen/anywhere-переносы для H1.
4. Кнопки защищены от перекрытия декоративной 3D-сценой: hero visual теперь `pointer-events: none`, текстовый слой hero поднят выше по z-index, интерактивные элементы явно сохраняют pointer events.

Ничего лишнего от себя не добавлялось.
