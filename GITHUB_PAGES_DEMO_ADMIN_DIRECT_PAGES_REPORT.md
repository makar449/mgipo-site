# v8.16 — Direct demo admin pages

Демо-админка исправлена так, чтобы разделы открывались как отдельные страницы, а не как переключение внутри одного экрана.

## Добавлено

- Явные статические страницы `/admin/dashboard/`, `/admin/home/`, `/admin/services/`, `/admin/categories/`, `/admin/documents/`, `/admin/legal/`, `/admin/navigation/`, `/admin/footer/`, `/admin/contacts/`, `/admin/leads/`, `/admin/pages/`, `/admin/seo/`, `/admin/settings/`, `/admin/users/`, `/admin/audit/`.
- Убран динамический route `/admin/[section]`, чтобы GitHub Pages не путал прямые страницы с динамической заготовкой.
- Сайдбар админки теперь навигирует на отдельные страницы, без локального переключения состояния через `onClick`.
- Убрана кнопка “Открыть вкладку” из шапки раздела, потому что пользователь уже находится на прямой странице.
- Убраны формулировки про предпросмотр в demo-admin UI.

## Важно

Это демо-админка для GitHub Pages. Сохранение работает через localStorage на устройстве показа. После подключения Supabase эти страницы можно перевести на реальную CMS.
