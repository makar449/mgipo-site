# V8.19 — Admin Deep Demo Panel

Что сделано:

- Админка оставлена на реальных маршрутах `/admin/...`, без preview-переключателя.
- Каждый раздел получил отдельную рабочую страницу: дашборд, главная, программы, категории, документы, правовые страницы, навигация, футер, контакты, заявки, страницы, SEO, настройки, пользователи, журнал.
- Добавлены редактируемые таблицы и карточки для программ, категорий, документов, навигации, заявок, SEO, пользователей и правовых текстов.
- Добавлено локальное сохранение структуры админки в `localStorage` для демонстрации заказчику на GitHub Pages.
- Главная, контакты и футер продолжают применять часть правок на публичной части через demo runtime.
- Добавлен журнал действий: изменения в разделах фиксируются в audit-ленте.
- Добавлены фильтры, поиск, добавление новых записей, смена статусов и inline-редактирование.

Ограничение демо-версии:

- Это публичная GitHub Pages-версия без сервера. Изменения сохраняются на устройстве в браузере. Для общей синхронизации всех пользователей следующим этапом нужен Supabase.
